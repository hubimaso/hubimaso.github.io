注册命令 非线程安全，建议集中在主线程中注册

参数
[in]node_name节点名称， 自定义节点名
[in]node_type节点类型，CMD_VIEW_NODE 或 CMD_ENABLE_NODE
[in]print_func命令执行函数，包含三个参数argc/argv/retbuf，用户只需要向retbuf中写入希望打印的字符即可 retbuf的最大长度是CMD_BUF_LEN argc和argv分别表示变量的个数和变量的值，注意是变量 比如命令，delete rule ID，只有一个变量就是ID，所以argc = 1, argv[0] = str(ID)
[in]cmd_str命令名称 命令由引导字符串(小写) + 变量(大写)组成，例如: ip address A.B.C.D/M label LINE ip address label为引导字符串，A.B.C.D/M为变量1，LINE为变量2 以下是几种常见的变量表示方式: WORD 一般变量 [WORD] 可选填变量，可以不填 <a-b> 带范围的数字变量，如<0-35791> A.B.C.D/M ipv4地址 X:X::X:X/M ipv6地址 (a|b|c) 多选一变量，如(on|off)
[in]doc_str帮助文档 每一行修饰一个引导字符串或变量，注意doc_str是一个完整的字符串，不要加"," 命令cmd_str: "ip address A.B.C.D/M label LINE", 文档doc_str: "Interface Internet Protocol config commands\n" "Set the IP address of an interface\n" "IP address (e.g. 10.0.0.1/8)\n" "Label of this address\n" "Label\n");
返回
成功：0，失败：-1
警告
参数不能为空
