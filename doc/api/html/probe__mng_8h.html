◆ PROBE_START

#define PROBE_START (   probe_id,
  probe_name,
  thread_id 
)

函数探测起始点

参数
[in]probe_id在函数内部唯一(与PROBE_END的probe_id保持一致)
[in]probe_name在函数内部唯一，探测点名称(与PROBE_END保持一致)
[in]thread_id如果是单线程，thread_id转入0，多线程传入线程id(0~n)
警告
参数不能为空
注解
PROBE_START和PROBE_END成对出现，可以在一个函数内探测多个点，而FUNC_PROBE_START和FUNC_PROBE_END只能探测函数
