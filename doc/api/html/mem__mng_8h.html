◆ dpdk_mempool_create()

struct rte_mempool* dpdk_mempool_create ( const char *  name,
unsigned  n,
unsigned  elt_size,
int32_t  numa_id,
unsigned  flags 
)

dpdk内存池创建

参数
[in]name内存池名称，不能重名
[in]n内存池大小
[in]elt_size内存节点大小，仅支持固定长度
[in]numa_id要申请的内存在numa_id对应的内存上申请，使用者可以通过get_numaid_by_cpuid获取numa_id 即使用者在对应cpu上操作numa_id上的内存。
[in]flags0:表示多生产多消费； MEMPOOL_F_SP_PUT|MEMPOOL_F_SC_GET 表示单生产单消费； MEMPOOL_F_SC_GET 表示多生产单消费； MEMPOOL_F_SP_PUT 表示单生产多消费；
返回
成功：内存池指针，失败：NULL
警告
参数不能为空
参见

示例:
